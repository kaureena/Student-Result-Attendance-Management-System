# Data README (Provenance and Synthetic Data Policy)

This repository uses synthetic/anonymised student data for portfolio demonstration.
No real student identities or personal data should be stored.

Data locations:
- v1 SQLite demo DB: v1_bca_basic_system/database/local.db
- v1 exports: v1_bca_basic_system/export/*.xlsx
- v2 data zones: v2_analytics_modernisation/data/raw|staged|curated
- v2 reports: v2_analytics_modernisation/reports

Synthetic rules:
- Fictional names and roll numbers only.
- No addresses, phone numbers, DOBs, emails, or identifiers.
- Keep datasets minimal but realistic for analysis.
