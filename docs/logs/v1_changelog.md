# v1 Changelog — Baseline (BCA-style)

Window (tentative): 01 Feb 2009 → 15 March 2009

## v1.0 (baseline freeze) — 15 March 2009
### Added
- Students module (CRUD), unique roll number validation.
- Attendance module (one record per student per date) + summary.
- Results module (marks entry, totals, pass/fail).
- Sample exports (attendance and results spreadsheets).
- UI screenshots demonstrating the full flow.

### Changed
- Standardised export columns and file naming for reviewer readability.

### Fixed
- Prevented duplicate attendance and duplicate result rows per student/subject/term.
