# Student Result & Attendance Management System

## Project Level
BCA Academic Project

## Project Background

This project was originally developed in **2009** as part of the **BCA academic curriculum**.
At that time, the system was implemented using basic C programming and file handling concepts.

This repository contains a **recreated and documented version** of the original academic project,
uploaded later for learning, reference, and documentation purposes

## 1. Project Introduction
The Student Result & Attendance Management System is a simple academic project designed to manage student records, attendance, and examination results using structured programming and relational database concepts.

The objective of this project is to apply fundamental concepts of:
- Logic development
- Database management
- Data validation
